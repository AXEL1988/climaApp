Instructions 

---------------------------------------------------------------------------------------------
1.- npm init --yes
2.- npm i webpack webpack-cli style-loader css-loader webpack-dev-server html-webpack-plugin

    * npm i webpack --save-dev -> Install webpack 
    * npm i webpack-cli --save-dev => Install cliente webpack ya que no trabaja solo
    * style-loader css-loader -> Son complementos que permite cargar css en js.
    * webpack-dev-server -> Ambiente de desarrollo
    * html-webpack-plugin ->Para convertir y copiar mi html a la carpeta dist

3.- Install bootstrap
    * Voy a https://bootswatch.com/4/lux/bootstrap.min.css
4.- Create index.html
5.- Create webpack.config.js
